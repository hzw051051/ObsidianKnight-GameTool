# 模板图片目录

此目录用于存放游戏界面的模板图片，用于图像识别。

## 需要的模板

运行程序后，请从游戏中截取以下按钮/元素的图片并保存到此目录：

| 文件名 | 说明 | 来源界面 |
|--------|------|----------|
| `btn_start.png` | "开始"按钮 | 关卡准备界面 (图1) |
| `btn_continue.png` | "继续"按钮 (圆形播放图标) | 障碍物界面 (图3) |
| `btn_retry.png` | "重试"按钮 | 胜利界面 (图5) |
| `card_frame.png` | 卡牌边框 (可选) | 卡牌选择界面 (图2) |
| `choice_box.png` | 选项框 (可选) | 障碍物选择界面 (图4) |

## 截取说明

1. 使用 ADB 或模拟器自带的截图功能
2. 用图像编辑软件裁剪出按钮区域
3. 保存为 PNG 格式
4. 文件名必须与上表一致

## 注意事项

- 如果没有模板图片，程序会使用固定坐标点击
- 模板匹配更准确，建议尽可能提供模板
- 模板图片分辨率应与模拟器一致 (1600x900)
